# Contributing to the NBA Predictions Game

First off, thank you for considering contributing to the NBA Predictions Game! We welcome contributions from the open-source community.

## How to Contribute

To contribute, please follow these steps:

1.  **Fork the repository** and create a new branch for your changes.
2.  **Follow the code style** guidelines listed below.
3.  **Write tests** for your changes.
4.  **Update the documentation** as needed.
5.  **Open a pull request** and request a review.

## Code Style

-   **Python**: Follow the [PEP 8](https://www.python.org/dev/peps/pep-0008/) style guide.
-   **JavaScript/TypeScript**: Follow the [Airbnb style guide](https://github.com/airbnb/javascript).

## Testing

-   **Backend**: Use `pytest` for unit and integration tests.
-   **Frontend**: Use Jest and React Testing Library for unit and component tests.

## Documentation

We have a comprehensive set of documentation to help you get started. You can find it in the `docs/` directory.

-   [Project Overview](project_overview.md)
-   [Backend Documentation](backend/README.md)
-   [Frontend Documentation](frontend/README.md)
-   [Deployment Documentation](./docs/deployment.md)

If you make any changes to the codebase, please update the documentation accordingly.

## Questions?

If you have any questions, please feel free to open an issue on GitHub.
